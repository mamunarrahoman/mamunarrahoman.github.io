PROJECT TITLE: Home Automation using ESP-32 (ESP8266) with integrated oscilloscope using Arduino Nano(ATmega 328P) to monitor signal quality of power grid.

Project duration: Undergraduate, 3rd Year 2nd Semester.

Objective: The objective of this project was to design and implement a comprehensive IoT-based Home Automation System capable of controlling and monitoring various household electrical loads such as fans, lights, and air conditioners through a mobile application. The system was developed to support ON/OFF control, task scheduling, and voice-command operation.

As part of the implementation, we integrated a temperature and humidity sensor to continuously monitor environmental conditions and display the readings on an OLED display. The fan was also configured to operate automatically based on predefined temperature thresholds.

To ensure reliable load handling, we implemented a mini-oscilloscope module that monitors the incoming power-grid signal quality (to check system is harmonics free and exactly 50 Hz or it’s fluctuating time to time).

Tool used: Arduino IDE, EasyEDA

Programming Language used: C++

IoT protocol used: SinricPro

Figure: Overall outlook of IoT based home automation system with integrated oscilloscope for monitoring incoming power from power grid.

Abstract: This project was implemented using the SinricPro IoT protocol to enable remote control of household electrical loads through a mobile phone or any internet-connected smart device. The system utilized the SinricPro library, which we customized according to our requirements and the number of loads. SinricPro provides free control for up to three loads, and additional loads require a subscription.

To enhance user interaction, the system was also integrated with Google Home, enabling complete voice-controlled operation and access to all Google Home features for seamless home automation.

Environmental monitoring was achieved using a DHT11 sensor for temperature and humidity measurements, with real-time data displayed on an OLED screen. Additionally, a mini-oscilloscope module was integrated to observe the incoming power-grid signal quality and frequency, using code adapted from open-source Arduino community resources.

Overall, the project successfully demonstrates a robust, customizable, and voice-enabled IoT-based Home Automation System.

Difficulties Faced: The main difficulty we faced was during the implementation of the oscilloscope circuit on the veroboard, as it was picking up a significant amount of electrical noise, which affected the signal quality and stability. To solve this problem we maintained efficient routing on veroboard.

  • Implemented the complete system, including hardware connections, module integration, and testing.

  • Ensured smooth integration of IoT functionalities, sensor modules, OLED display, and mini-oscilloscope into a fully working home automation system.
